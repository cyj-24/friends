# 🚐 房车环游中国 - 虚拟Agent直播社区

一个为房车旅行团队设计的实时状态分享系统。四位旅行者（JCL、PYJ、LWJ、CYJ）可以通过这个平台，让关注者实时了解他们「现在在哪、在干什么、在吃什么」。

## 🌟 核心功能

### 1. 🗺️ 实时位置地图
- 在地图上实时显示房车位置
- 点击地图更新当前城市
- 自动绘制旅行轨迹路线
- 显示已访问城市标记

### 2. 👥 四人Agent状态面板
每位团队成员的实时卡片显示：
- **位置**：当前所在城市
- **活动**：正在做什么（开车、露营、吃饭等）
- **餐饮**：正在吃什么
- **心情**：当前心情状态
- **能量**：体力值百分比

### 3. 📜 旅行日志时间轴
- 自动记录位置更新、状态变更
- 支持手动发布图文动态
- 按时间倒序展示
- 显示发布地点

### 4. 📊 旅行数据统计
- 旅行天数计数
- 累计行驶里程
- 经过省份数量
- 当前位置显示

### 5. 👀 双模式支持

**团队成员模式**：
- 切换身份（JCL/PYJ/LWJ/CYJ）
- 更新自己的实时状态
- 发布旅行日志
- 点击地图更新位置

**关注者模式**（URL后加 `?view`）：
- 只能查看，不能编辑
- 实时观看四人状态
- 适合分享给粉丝/家人/朋友

## 📱 使用说明

### 团队成员使用

1. **打开页面**，点击顶部切换自己的身份
2. **更新状态**：
   - 在右侧自己的卡片中输入当前活动
   - 输入正在吃什么
   - 选择当前心情
   - 点击「更新状态」
3. **更新位置**：
   - 点击地图上的新位置
   - 输入城市名称
   - 自动记录到路线和日志
4. **发布日志**：
   - 点击右下角「+」按钮
   - 输入文字内容
   - 点击发布

### 分享给关注者

将链接加上 `?view` 参数分享给关注者：
```
https://你的域名.com/?view
```

关注者只能查看，不能编辑任何内容。

## 🗺️ 技术架构

- **前端**：纯 HTML + CSS + JavaScript
- **地图**：Leaflet.js + OpenStreetMap
- **数据存储**：LocalStorage（本地持久化）
- **实时同步**：定时轮询（每10秒自动刷新）
- **部署**：支持 GitHub Pages / Netlify / Vercel

## 💡 玩法建议

### 日常更新
- 每天更新几次位置和状态
- 吃到好吃的拍照+文字记录
- 遇到有趣的风景及时标记位置

### 里程碑记录
- 每到达一个新省份庆祝一下
- 行驶每满1000公里发日志纪念
- 记录每天的里程和花费

### 与关注者互动
- 发布「猜猜我们明天去哪」互动
- 分享当地美食推荐
- 回答关注者的留言提问
